Posting to GitHub in case others scripting in RE might find this useful. 


## Usage

Copy the "Eremite" folder to your RazorEnhanced/Scripts/ directory so it is at <RazorEnhanced>/Scripts/Eremite so that imports work.

Edit the `_eremite.py` file or make your own to add any custom overrides, etc and make sure it's set to load at startup.
